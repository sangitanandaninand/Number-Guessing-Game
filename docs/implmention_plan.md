# Implementation Plan

## Feature 1: Guess the Number
**Trigger:** The player enters a guess.
- The program will check if the guess is too high or too low.
- It will continue to tell the player whether the guess is high or low until the number is figured out.

---

## Feature 2: Count the Guesses
**Trigger:** Every time the player makes a guess.
- It will count how many guesses the player makes.
- At the end, the program will display the total number of guesses used.

---

## Feature 3: Play Again
**Trigger:** After the game is over.
- After the game is over, the program will ask the player if they want to play again or play with a new random number.

 ## Feature 4: Best Score
 ** Trigger:** After the game ends.
 - The program will save the user's best record, which is the fewest number of guesses that is use for to win.
 - It wil display the best record at the end of each game.
